# TV-Data-Scanner
A lightweight DOM scanner for OHLC data extraction.
# TV-Data-Scanner v4.1

基于前端 DOM 监听技术的数据提取工具。

## 核心特性
- **穿透技术**：支持嵌套 Iframe 框架的数据检索。
- **逻辑排重**：基于价格极值锚定，消除重复采样。
- **全周期支持**：兼容从 M15 到 Daily 的所有时间周期。

## 使用说明
1. 打开控制台 (F12)。
2. 粘贴 `index.js` 中的代码并回车。
3. 在图表上匀速移动鼠标，自动捕获 30 根 K 线数据。
